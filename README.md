DAMAR PORTFOLIO — CINEMATIC WEB (FINAL)

1. Masukkan semua SVG hasil export PDF ke folder:
   assets/pages/

2. Rename file jadi berurutan:
   page-01.svg sampai page-29.svg

3. Buka index.html untuk preview.

4. Upload semua file ke GitHub repository.

5. Aktifkan GitHub Pages:
   Settings > Pages > Deploy from a branch > main > /(root)

Catatan:
- File ini sudah siap untuk GitHub Pages.
- Kalau ingin edit kontak, buka index.html lalu cari section id="contact".
- Kalau ingin ganti judul section atau urutan halaman, edit PORTFOLIO_CONFIG di bagian bawah index.html.
